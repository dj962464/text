Html总结

元素可以包含文本内容和其他元素，也可以是空的

1.包含文本内

2.元素嵌套：形成更为复杂的语法

属性和值：属性是用来修饰元素的标记 属性="值" 属性="值";

文档结构:

包含两个特殊的标签head,body

body中的内容就是浏览器解析的内容

常用标签:

超链接：设置页面中允许被点击的内容。

段落标签:段落是通过"/p"标签定义的

标题标签:标题是通过 h1 - h6; 标签进行定义的。

图像标签:由"/img"标签定义，要在页面上显示图像,你需要使用源属性src 源属性的值是图像的 URL 地址

列表：无序列表是一个项目的列表，此列项目使用粗体圆点（典型的小黑圆圈）进行标记.

无序列表始于"p"标签。每个列表项始于"li"

同样，有序列表也是一列项目，列表项目使用数字进行标记。有序列表始于ol标签。每个列表项始于li标签

表单元素form元素 HTML 表单用于收集用户输入。input元素是最重要的表单元素。input元素有很多形态，根据不同的 type 属性

CSS总结

选择器:选择器是您需要改变样式的对象。每条声明由一个属性和一个值组成。（无论是一条或多条声明，都需要用{}包裹，且声明用;分割）

属性（property）是您希望设置的样式属性（style attribute）。每个属性有一个值。属性和值被冒号分开。

元素选择器:元素选择器根据元素名称来选择 HTML 元素,所有的p标签文本内容和颜色都会相应变化。

id 选择器：id 选择器使用 HTML 元素的 id 属性来选择特定元素。

元素的 id 在页面中是唯一的，因此 id 选择器用于选择一个唯一的元素！

要选择具有特定 id 的元素，请写一个井号（＃），

类选择器:类选择器选择有特定 class 属性的 HTML 元素。

css的使用:

外部样式表：使用link标签引入外部css文件，推荐使用

内部样式表：在head标签中引入了style标签

内联样式：所谓内联样式，就是直接把样式规则直接写到要应用的元素中

级联优先级从高到低分别是： 内联样式 内部样式表或外部样式表

css定位:position 属性规定应用于元素的定位方法的类型。
